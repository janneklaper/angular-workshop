Assignment 1: List the whiskies

  • Find the file src/WhiskyList/WhiskyList.html and notice the table-element with ng-repeat
  • Discover the WhiskyListCtrl.js-file under the same folder and notice how it's tied to the mentioned view in src/appRoutes.js
  • Notice the WhiskyService.js also in the same folder. This will be the service that provides us with a JSON-file full of whisky
  • In WhiskyListCtrl, notice that the WhiskyService is already injected. Call the WhiskyService, insert the whiskies into the scope and observe as the whisky will start flowing into your screen when you refresh the page
Related documentation: NgRepeat

Bonus assignment: Stop the copy paste

  • Directives provide a clever way of reusing similar functionality thoughout the application. This removes the need to copy similar elements between views and makes the application more modular.
  • Implement a directive that replaces a single row in the whisky list. You may utilize the template file located at src/WhiskyList/WhiskyListItemDirective.js
  • Observe that the directive is named 'listItem', but when you use it as an element, the correct way to reference it is with 'list-item'
Related documentation: Directive
